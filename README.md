# Tarot Today PWA

간단한 타로 오늘의 운세 PWA 앱입니다.

## GitHub Pages 배포 방법

1. GitHub에서 새 저장소를 만듭니다. (예: `tarot-today`)
2. 이 폴더 안의 모든 파일을 그대로 저장소에 업로드합니다.
3. GitHub 저장소에서 **Settings → Pages**로 이동합니다.
4. Source를 `GitHub Actions`로 선택하고, 기본 설정으로 저장합니다.
5. 제공된 GitHub Actions 워크플로우(`.github/workflows/gh-pages.yml`)가 실행되면
   `https://USERNAME.github.io/REPO` 주소로 접속할 수 있습니다.

모바일 크롬에서 해당 주소를 열고, 브라우저 메뉴에서 **"홈 화면에 추가"** 또는
**"Install app"** 메뉴를 선택하면 PWA처럼 설치됩니다.
